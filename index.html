ERP: Enterprise Resource Planning - (DAY 1)

Enterprise resource planning is software that use to manage and track day-to-day business activities such as accounting, project management, risk management, supply chain operations, etc.

Major reason for choosing ERP
  1. Integrate financial information: generate the trade report, profit and loss report.
  1. Customer order information.
  1. Standardize and speed up the operation process.
  1. Reduce inventory: the process of lowering inventory levels to a point where they meet customer demand.
  1. Standardize HR information: maintain all information.

AOL Components - (DAY 3)

AOL also known as Foundation (FND) module

  • Lookup

    Lookups are lists of values in applications. You define a list of values as a lookup type consisting of a set of lookup codes, each code's translated meaning, and optionally a tag. End users see the list of translated meanings as the available values for an object.

  • Value Set

    A value set in Oracle Applications is a list of values with validations. It's used to restrict the values a user can enter. For example, when submitting a concurrent program, you can associate a value set to a parameter to ensure the user only enters valid values.

  • Message

How to submit xml publisher report from PLSQL:

  • errbuf to return any error messages, and retcode to return completion status.
  • The parameter retcode returns 0 for success, 1 for success with warnings, and 2 for error.
  • fnd_request.ADD_LAYOUT
    fnd_request.SUBMIT_REPORT

    retcode - 0 - normal, 1 - warning, 2 - error

    unix scripting - 0 normal, 1 - error, 2 - warning, 3 - fatal error

Steps:

  • create procedure (DCS_VEN_INFO_PRC)
  • submit_request - Oracle seeded packaged procedure, one of the API given and provided by oracle to submit a request
  • package specification - submit_request

Multiple language using xml report

Oracle gives a very straightforward method to create multilingual reports if we use XML Publisher to design the reports. We no longer need to create very different report layouts for the same report to be used across languages. Once a language has been installed in addition to the default language, English, then we only need to assign the layout to the language and the characteristics of the language will determine how the output will look like.
